Live Studies Application Page
=============================

Page URL after deployment:
/live-studies-application

Form submission:
- Submissions are sent via POST to a Google Apps Script web app.
- The script appends each application to the "Applications" sheet in the
  "FinXT Live Study Applications" Google Spreadsheet.
- Redirect after submission: /thank-you?source=live-studies

Google Apps Script setup:
1. Open the FinXT Live Study Applications Google Sheet.
2. Extensions → Apps Script.
3. Deploy as a web app (Execute as: Me, Who has access: Anyone).
4. Copy the /exec URL into pages/live-studies-application.js (GOOGLE_SCRIPT_URL).
5. After script changes: Deploy → Manage deployments → New version → Deploy.

Daily email summary:
- Function: sendDailyApplicationSummary
- Sends a daily summary to finxt.info@gmail.com
- Set up via Apps Script → Triggers → Time-driven → Day timer

How to view applications:
1. Open the FinXT Live Study Applications Google Sheet.
2. Go to the Applications tab.
3. Export or filter as needed in Google Sheets / Excel.

Security (form + Apps Script):
- Client: maxLength on text fields, honeypot (website), hidden form_secret.
- Server: see google-apps-script/live-studies.gs
  - sanitizeCell() on all fields (trim, max length, formula-prefix escape)
  - Email + phone validation, country allowlist, terms required
  - Honeypot + FORM_SECRET check (must match lib/liveStudiesFormSecurity.js)
- After updating the .gs file: Deploy → Manage deployments → New version → Deploy
- Deploy the website AFTER the script is redeployed (secret must match).

Important:
- Localhost and production both submit to the same Google Sheet.
- The thank-you page always appears after submit; confirm new rows in the sheet.
- FORM_SECRET is visible in the site bundle (deters casual spam, not determined attackers).
- For heavy bot spam, add reCAPTCHA later.
